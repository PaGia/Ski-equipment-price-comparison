# 產品需求文件 (PRD)

## 專案名稱
雪板價格比較器 (Ski Equipment Price Comparison)

## 專案概述
一個網頁應用程式，自動從多個網路商店抓取雪板商品資料，並提供價格比較功能，幫助使用者找到最優惠的價格。

## 目標用戶
- 雪板愛好者
- 想購買雪板設備的消費者
- 希望跨國比價的買家

## 核心功能

### 1. 多店家商品爬取
- 支援內建店家（Murasaki Sports、North Shore）
- 支援動態新增自訂店家
- 支援多種電商平台（BASE、Shopify、通用網站）
- 定時自動更新（每日 06:00 JST）

### 2. 價格比較
- 跨店家價格對比
- 多幣別支援（JPY/CAD/USD/EUR/GBP/AUD/TWD）
- 自動匯率轉換（統一換算為日圓）
- 顯示最低價格與來源店家

### 3. 商品管理
- 品牌識別與分類
- 商品名稱標準化（去除年份、尺寸）
- 同款商品自動合併

### 4. 使用者介面
- 商品搜尋與篩選
- 品牌篩選
- 價格範圍篩選
- 網格/列表檢視切換

## 非功能需求
- 爬蟲需處理 JavaScript 渲染頁面（Puppeteer）
- 價格驗證機制（過濾異常價格）
- 錯誤處理與重試機制

## 未來擴展方向
- 價格歷史追蹤
- 降價通知功能
- 更多店家支援
- 用戶收藏清單
